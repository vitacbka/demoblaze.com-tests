# 🧪 Demoblaze Test Automation![img_1.png](img_1.png)

Automated UI tests for [Demoblaze](https://www.demoblaze.com) using **Java + JUnit 5 + Selenide**.

## 🛠️ Tech Stack
- Java 11+
- JUnit 5
- Selenide
- Maven
- Chrome / Firefox

## ▶️ Run Tests

```bash
# Chrome (default)
mvn test

# Firefox
mvn test -Dbrowser=firefox
